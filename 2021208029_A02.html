IntelliJ IDEA 단축키 15가지

제가 맥북을 사용해서 MacOS 기준으로 작성합니다.

  • ⌥⏎

    문제에 대한 수정 제안 목록 표시

  • F2

    다음 오류, 경고, 제안으로 이동

  • ⌘1

    프로젝트 창을 열고 초점 이동

  • esc

    에디터로 다시 초점 이동

  • ⌘E

    최근에 열었던 파일 확인

  • ⌘B

    심볼의 선언으로 이동

  • ⌥F7

    어떤 항목이 사용된 위치를 검색

  • Double ⌃

    Run Anything 창이 열림

  • ⌥↑ / ⌥↓

    커서 근처의 코드 선택 영역을 확대하거나 축소

  • ⌘/

    해당 코드 줄을 주석으로 처리

  • ⇧⌘⏎

    현재 구문 자동 완성

  • ⌥⌘L

    현재 파일의 서식을 지정

  • ⇧⌘A

    모든 액션을 검색

  • ⌃T

    사용 가능한 리팩토링 옵션 표시

  • Double ⇧

    모든 검색
