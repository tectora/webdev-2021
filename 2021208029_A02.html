⌘1 프로젝트 창을 열고 초점 이동
esc 에디터로 다시 초점 이동
⌘E 최근에 열었던 파일 확인

⌘/ 해당 코드 줄을 주석으로 처리

⌥⌘L 현재 파일의 서식을 지정
⇧⌘A 모든 액션을 검색
